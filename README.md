# Chhavi's Portfolio

A clean, modern portfolio website showcasing Chhavi's work and projects.

## 🌐 Live Demo

[https://chhaavii.github.io/dyportfolio.io/](https://chhaavii.github.io/dyportfolio.io/)

## ✨ Features

- **Clean Design**: Minimal, modern interface with smooth animations
- **Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Elements**: 
  - Hover effect on "PORTFOLIO" text revealing "CHHAVI"
  - Gradient social media buttons
  - Smooth transitions and micro-interactions
- **Fast Loading**: No heavy frameworks or dependencies
- **SEO Optimized**: Proper meta tags for social sharing

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with animations
- **JavaScript**: Interactive features and smooth scrolling
- **Google Fonts**: Typography (Inter, Manrope, Antonio, Archivo Black)

## 📱 Responsive Design

The portfolio is fully responsive and adapts to different screen sizes:

- **Desktop**: Full experience with large typography and spacing
- **Tablet**: Optimized layout for medium screens
- **Mobile**: Compact design with adjusted font sizes and button layouts

## 🎨 Design Elements

- **Color Scheme**: Clean white background with gradient accents
- **Typography**: Mix of modern fonts for hierarchy and readability
- **Animations**: Smooth hover effects and transitions
- **Layout**: Centered content with fixed social links

## 🚀 Getting Started

To run this portfolio locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/chhaavii/dyportfolio.io.git
   ```

2. Navigate to the project directory:
   ```bash
   cd dyportfolio.io
   ```

3. Open `index.html` in your web browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```

4. Visit `http://localhost:8000` in your browser

## 🔧 Customization

### Changing Personal Information

Edit the following sections in `index.html`:

- **Social Links**: Update the `href` attributes in the social buttons section
- **Email**: Change `mailto:chhavi@example.com` to your actual email
- **Social Profiles**: Update LinkedIn, GitHub, and website URLs

### Styling

All styles are contained in the `<style>` section of `index.html`. You can modify:

- **Colors**: Update CSS variables in the `:root` selector
- **Fonts**: Change font families or import different Google Fonts
- **Animations**: Adjust transition durations and effects
- **Layout**: Modify flexbox and grid properties

## 🌟 Features Breakdown

### Portfolio Text Effect

The main "PORTFOLIO" text has an interactive hover effect:
